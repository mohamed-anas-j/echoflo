# Contributing to EchoFlow

Thank you for your interest in contributing to EchoFlow! We welcome contributions from the community.

## Getting Started

1. **Fork the repository** on GitHub
2. **Clone your fork** locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/echoflo.git
   cd echoflo
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Create a new branch** for your feature or fix:
   ```bash
   git checkout -b feature/your-feature-name
   ```

5. **Run the development server**:
   ```bash
   npm run dev
   ```

## Development

- **TypeScript**: This project uses TypeScript. Please maintain type safety.
- **Code Style**: Follow the existing code style and conventions.
- **React**: We use React 18. Functional components and hooks are preferred.

## Submitting Changes

1. **Commit your changes** with clear, descriptive messages:
   ```bash
   git commit -m "feat: add new feature description"
   ```

2. **Push to your fork**:
   ```bash
   git push origin feature/your-feature-name
   ```

3. **Create a Pull Request** on GitHub with:
   - Clear title describing the change
   - Description of what and why
   - Any relevant issue numbers
